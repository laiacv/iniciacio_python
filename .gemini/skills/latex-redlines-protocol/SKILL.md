# Latex Redlines Protocol

This skill ensures that all interactions and modifications follow a strict set of safety and quality rules designed for high-standard research work.

## REDLINES & Interaction Protocols

These rules have absolute precedence and must be followed for every task.

### 1. Strict Modification Policy
- **DO NOT MODIFY ANYTHING** that has not been explicitly requested. This includes both the style and content of any part of the document.
- Modifications are limited EXCLUSIVELY to the elements or sections indicated by the user.
- **The Preamble is untouchable**, unless an explicit modification is requested.
- **REDLINE: NEVER ENTER the `avans_3rTrim` folder** unless explicitly requested in text.

### 2. Research and Understanding Phase
- Before making any changes, analyze and understand the document structure.
- Identify which elements, packages, and custom commands are available in the preamble to use them coherently in content generation.

### 3. Prior Validation (TO DO Protocol)
- Before applying changes (especially structural or significant content changes), generate a task list (**TO DO**).
- This list must be **validated by the user** before executing any action.
- This protocol is mandatory if changes involve different project areas or related files.

### 4. Style and Consistency
- Always maintain the format, language, and conventions used in the original document to ensure seamless integration.
- **Research Standard:** Aim for a beautiful, plain, standard research work style. The output should be pleasant and reflect high effort and level.

## LaTeX Best Practices for Research

When generating LaTeX content:
- Use clear and organized structures.
- Ensure mathematical notation is consistent.
- Follow the established aesthetic: elegant and professional.

## Workflow

1. **Analyze:** Read the existing files to understand the context.
2. **Plan (TO DO):** Propose the changes in a TO DO list.
3. **Wait:** Do not proceed until the user approves the TO DO list.
4. **Execute:** Apply the approved changes surgically.
5. **Validate:** Check for consistency and style.
